# Sistem Analisis PBD - SMK Aminuddin Baki

Fail utama untuk GitHub Pages ialah `index.html`.

## Cara publish di GitHub Pages
1. Cipta repository baharu di GitHub.
2. Muat naik fail `index.html` ke repository tersebut.
3. Pergi ke **Settings > Pages**.
4. Pilih **Deploy from a branch**.
5. Pilih branch `main` dan folder `/root`.
6. Klik **Save**.

Sistem ini telah dikemas kini untuk:
- SEKOLAH MENENGAH KEBANGSAAN AMINUDDIN BAKI
- Apps Script URL dan CSV baharu
- Lencana sekolah baharu
