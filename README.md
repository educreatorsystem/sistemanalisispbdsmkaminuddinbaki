Fail utama untuk GitHub Pages ialah `index.html`.

## Cara publish di GitHub Pages
1. Buka repository GitHub.
2. Upload fail `index.html` ke root repository.
3. Pergi ke Settings > Pages.
4. Pada Branch, pilih `main` dan folder `/root`, kemudian Save.
5. Tunggu sehingga pautan GitHub Pages aktif.
