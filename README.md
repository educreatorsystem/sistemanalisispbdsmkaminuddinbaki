# Sistem Analisis PBD - SEKOLAH MENENGAH KEBANGSAAN AMINUDDIN BAKI

Fail utama untuk GitHub Pages ialah `index.html`.

## Cara publish di GitHub Pages

1. Cipta repository baharu di GitHub.
2. Muat naik fail `index.html` ke repository tersebut.
3. Pergi ke **Settings > Pages**.
4. Pada bahagian **Build and deployment**, pilih:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Klik **Save** dan tunggu pautan GitHub Pages dijana.
